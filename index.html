Welcome to Driji.

Driji is an attendance based fingerprint application for school.

Features :

Here are some of the features that we think that you'll love.

Centralized : You can manage multiple attendance machine in single application.

Low Cost : No need more hardware, the only single server and one attendance machine or more, no other device need to configure.

Easy To Use : Driji has always been easy to use for any user.

SMS Gateway : Integrated with sms gateway, the system able to send notify to all member.
